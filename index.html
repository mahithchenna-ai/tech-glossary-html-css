Core Terms
[24 definitions and counting!]

API

Application Programming Interface - a set
of rules that lets different software applications talk to each other.

Async/Await

JavaScript syntax for handling
asynchronous operations in a
synchronous-looking way, making code
easier to read.

Bandwidth

The maximum amount of data that can be transferred over a network connection in a given time period.

Cache

Temporary storage that saves frequently accessed data so it loads faster next time you need it.

CI/CD

Continuous Integration/Continuous Deployment - automated practices for testing and releasing code changes
quickly.

Container

A lightweight, standalone package that includes everything needed to run software consistently across different
environments.

Database

An organized collection of structured data stored electronically and accessed via a computer system.

DNS

Domain Name System - translates human-readable website names into IP addresses computers use to find each other.

Endpoint

A specific URL where an API can access the resources it needs to perform its function.

Framework

A pre-built foundation of code that developers use as a starting point to build applications faster.

Git

Version control system that tracks
changes in code and lets developers collaborate without overwriting each
other's work.

HTTP

HyperText Transfer Protocol - the set of
rules for transferring files like text and images on the web.

IDE

Integrated Development Environment - software that combines code editing, debugging, and testing tools in one place.

JSON

JavaScript Object Notation - a lightweight format for storing and transporting data between servers and web apps.

Kubernetes

Open-source platform for automating deployment, scaling, and management of containerized applications.

Latency

The time delay between a request being
sent and a response being received over a network.

Microservice

Architecture pattern where an application
is built as a collection of small,
independent services instead of one large program.

Node.js

JavaScript runtime that lets you run JavaScript code on servers instead of just in web browsers.

OAuth

Open standard for access delegation that lets users grant websites access to their
info without sharing passwords.

Postgres

PostgreSQL - powerful open-source
relational database system known for
reliability and robust features.

Query

A request for data or information from a database, written in a specific language
like SQL.

REST

Representational State Transfer - architectural style for designing
networked applications using simple HTTP
requests.

SDK

Software Development Kit - collection of tools, libraries, and documentation for building apps for a specific platform.

TypeScript

JavaScript with syntax for types - helps catch errors early by defining what kind of data variables should hold.
